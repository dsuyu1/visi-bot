# VISI Announcement Bot

Announces upcoming VISI competitions and events in Discord. Posts automatically at **7 days** and **1 day** before each event, and responds to manual commands.

---

## Commands

| Command | What it does |
|---|---|
| `!events` | Lists all upcoming events with IDs |
| `!announce all` | Same as `!events` |
| `!announce <id>` | Full embed for a specific event (e.g. `!announce ncl-fall26`) |
| `!schedule` | Shows what the bot will auto-post and when |
| `!help` | Command reference |
